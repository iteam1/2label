# cvat2via

tool convert cvat annotation `CVAT for images 1.1` to via-via 2.0 annotation `via_region_data.json`

**cvat**

        cvat
        ├── annotations.xml
        └── images
            ├── img2.jpg
            ├── img3.jpg
            └── img.jpg 

**via**

        via
        ├── img2.jpg
        ├── img3.jpg
        ├── img.jpg
        ├── via_region_data.csv
        └── via_region_data.json
